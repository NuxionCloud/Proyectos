## Cómo retomar un proyecto en otro PC

```bash
git clone https://github.com/NuxionCloud/Proyectos.git
cd Proyectos/bocatitos/_app
npm install
npm run dev
```

El `node_modules/` no se sube (cada PC lo regenera con `npm install`).
